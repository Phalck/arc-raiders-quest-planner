# Arc Raiders Quest Planner — Project Guide

Always start by reading `SESSION.md` to understand the current state, what
was built, and the account/deploy situation.

Key files:
- `index.html` / `app.js` / `styles.css` — the PWA
- `data/quests.json` — scraped quest data (100 quests)
- `scraper/scrape.js` — wiki scraper for refreshing data
- `SESSION.md` — full session memory and feature roadmap

Deploy: `git push && vercel --prod`
